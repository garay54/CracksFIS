Componentes del sistema

Resumen operativo para interpretar el dashboard y sus limites metodologicos.

Entradas

  • W: anchura media de grieta en mm. Anclada a SICT/IMT.
  • L: longitud del track en mm. Umbrales iniciales propios.
  • D: fraccion de pixeles de grieta en la mascara.
  • C: completitud de observacion; 1 completa, 0 truncada.

Arquitectura

  1. FIS 1 combina W y L para obtener SD.
  2. FIS 2 combina D y C para obtener SE.
  3. FIS 3 combina SD y SE para obtener SF.

Inferencia

Se usa una variante difusa suavizada con producto algebraico para antecedente y centroide ponderado de consecuentes para defuzzificacion.

Limites

El sistema es preliminar. L, D, C, reglas y recomendaciones requieren sensibilidad, calibracion de campo y criterio experto.

Inferencia y alcance metodologico

Tipo: sistema Mamdani jerarquico con tres sub-FIS.

Antecedente: producto algebraico. Defuzzificacion: centroide ponderado de consecuentes activados.

Respaldo: W conserva anclaje directo en umbrales SICT/IMT; L, D, C, reglas y recomendaciones se tratan como configuracion metodologica inicial pendiente de calibracion experta.

Salida operativa: SF se interpreta con cortes internos preliminares: leve < 35%, moderada 35-70% y critica >= 70%.
